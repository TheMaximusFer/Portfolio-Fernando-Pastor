# SisInf
Este repositorio contiene las diferentes prácticas de Sistemas de Información de Juan José Muñoz Lahoz, Fernando Pastor Peralta y Marcos San Julián Fuertes

El repositorio está organizado en carpetas teniendo cada una los contenidos a entregar en cada práctica:
 - practica1: contiene la entrega de la primera práctica.
 - practica2: contiene la entrega de la segunda práctica, además de las imágenes del prototipo realizado.
 - practica3: contiene la entrega de la tercera práctica, la cual incluye memoria y las clases VO, DAO así como un CRUD en java sencillo
